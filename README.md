# 🏛️ Выбери Свой Путь

**«Выбери Свой Путь»** — это интерактивная веб-платформа, превращающая изучение истории, географии и культуры в увлекательное виртуальное путешествие. Приложение объединяет в себе 360° панорамы, аудио-сопровождение, краеведческую информацию и глубокую систему геймификации.

## ✨ Ключевые возможности

* 🌍 **Иммерсивные 360° туры:** Интеграция с сервисом Panoraven для плавного просмотра панорам с поддержкой гироскопа на мобильных устройствах.
* 🎮 **Продвинутая геймификация:** * Динамическое начисление опыта (XP) в зависимости от сложности маршрута и прогресса.
    * Система уровней и уникальных рангов (от «Новичка» до «Легенды»).
    * Интерактивные викторины в конце каждого маршрута для закрепления знаний.
* 👤 **Личный кабинет исследователя (Профиль):**
    * Отслеживание летописи побед (пройденных маршрутов).
    * Система "Тайников" (избранные локации для быстрого доступа).
    * Смена визуального облика (аватара) и редактирование имени в истории.
* 🛡️ **Smart Offline Mode:** Встроенный connection manager, который отслеживает `ping` до сервера. При потере сети приложение элегантно блокирует UI стильным Glassmorphism-оверлеем, предотвращая потерю прогресса.
* 🎵 **Атмосферный звук:** Фоновое аудио-сопровождение, уникальное для каждой тематической ветки.
* 💎 **Премиальный UI/UX дизайн:** * Интерфейс в стиле тёмного матового стекла (Dark Glassmorphism).
    * Идеально плавные анимации перехода между страницами (без скачков и сдвигов).
    * Сложные CSS-градиенты и кастомные элементы управления (селекты, скроллбары).

## 🗺️ Доступные маршруты

На данный момент платформа включает три масштабные тематические кампании:
1.  **Многонациональная Самара** (Наследие предков, Скрытые тропы, Городские легенды).
2.  **Гастрономическое путешествие по Грузии** (Западная Грузия, Восточная Грузия, Горные регионы).
3.  **Тайны Великого Шелкового пути** (Звезда Востока, Забытые оазисы, Путь шелка).

## 🛠️ Технологический стек

* **Frontend:** HTML5, CSS3, Vanilla JavaScript (ES6+ Classes). Никаких тяжелых фреймворков — максимальная производительность и модульная архитектура (`app.js`).
* **Backend:** Node.js, Express.js.
* **База данных:** SQLite3 (асинхронные транзакции, реляционная структура для связей пользователей, прогресса и избранного).
* **Интеграции:** Panoraven API (Iframe 360° Embeds).
