# ChooseYourPath
🏛️ Выбери свой путь — иммерсивная веб-платформа для интерактивных 360° путешествий. Исследуйте историю Самары, гастрономию Грузии и тайны Шёлкового пути. Включает систему геймификации (XP, уровни, ранги), личный кабинет, викторины и надежную защиту от потери соединения. Построено на Node.js, SQLite и Vanilla JS.
